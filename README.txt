# Квест на день рождения

Панель управления квестом.  
В нее входит:  
1. Система для регистрации кодов.
2. Просмотр текущего счета монет.
3. Система воспроизведения фоновой музыки.
4. Подключение личного кабинета по числу участников (не ограничено, единственным ограничением является объем оперативной памяти управляющего устройства).

players = {
    "@Roma": 0,
    "@Rayan": 0,
    "@Timur": 0
}

В исходном коде три участника, но всегда имеется возможность дописать некоторое количество участников или удалить (см. код выше, что соотвествует строке 7-11 в исходном коде).
Во время проведения квеста  это сделать невозможно, только на этапе подготовки.
Квест можно запускать неограниченное количествто раз.
Подключение организатора и всех участников квеста необходимо обеспечить в одной сети (подсети, например wi-fi домашней зоны), мобильный интернет не поддерживается).
При запуске кода программа выводит:
* Running on  и URL-адрес, по которому должны войти организатор и участники.
При входе у участников открывается форма ввода логина, которые соотвествуют списку players = {...}, у организатора - @Admin. 
Для дополнительной безопасности вход Админа осуществляется с вводом пароля - KarimChik .
Администратору в личном кабинете доступна информация обо всех участниках по их никнеймам о сотоянии монет. 
Управление определено тремя кнопками: 
ALERT - воспроизводит звук тревоги, который сообщает о необходимости игроков собраться в точке начала квеста.
START - запускает фоновую музыку.
STOP  - останавливает фоновую музыку, которая дает возможность участнику сдать задание на проверку, получить новый бланк или заменить бланк задания.

Задания придумывает организатор квеста до его начала.  
В системе предусмотрены коды для получения некоторого количества монет, они распечатываются организатором до начала квеста.

Какие-то коды должны быть спрятаны (наклеены на предметах, вложены в книги, за картины и т.д.). 
Некоторые должны храниться у организатора. Их должно быть столько же, сколько и заданий.
Участник должен найти код или попросить задание у организатора.  
Полученные коды регистрируются в личном кабинете участника.
Если участник попросил задание, то организатор выдает ему бланк задания, в котором есть условие и поле для записи ответа.
Участник при получении бланка задания имеет право попросить еще один бланк задания или обменять его на другой.
Участник в любой момент игры имеет право сдать выполненный бланк задания, получить вердикт "Правильно" и взамен получить код.
Если "Неверно", то получить новый бланк задания.

# Запуск программы
Для запуска программы необходима установка Python 3.8 или выше, при установке на Windows выбрать опцию:
Add Python 3.x to PATH
На OS Linux Python уже установлен.
После установки Python откройте командную строку (или, в случае MacOS или Linux - Терминал) и введите:

pip install Flask>=2.0.0 pygame==1.9.6
Для запуска квеста откройте командную строку (или, в случае MacOS или Linux - Терминал) и введитe:

git clone https://github.com/salut-msg/karim-bd.git
cd karim-bd
python3 main.py
